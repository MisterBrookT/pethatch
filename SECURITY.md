# Security And Takedowns

PetHatch hosts community-submitted creative assets.

## Asset Takedowns

If you own rights to a pet asset and believe it should not be distributed here, open an issue with:

- pet id
- asset path or URL
- proof of ownership
- requested action

We will remove or disable the asset while reviewing the request.

## Runtime Safety

Pet packs should be static assets only:

- `pet.json`
- `spritesheet.webp` or `spritesheet.png`
- preview images or GIFs
- license/readme files

Do not submit executable code inside a pet pack. Runtime adapters and installers should treat pet packs as untrusted data.
